## About 66App

66App is a social content platform with features including a home feed, short-form video (Shorts), post creation, search, user profiles, notifications, and an affiliate programme. The Flutter client uses Riverpod for state management, GoRouter for navigation, and Dio with PASETO tokens for API communication.
